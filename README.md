# 🚗 CRUD Aluguel de Carros - Python

## 📖 Descrição
Este projeto é um sistema simples de gerenciamento de aluguel de carros, desenvolvido em Python, com foco na implementação das operações básicas de um CRUD (Create, Read, Update, Delete).

---

## 🎯 Objetivo
Aplicar conceitos fundamentais de programação, incluindo:
- Estruturas de dados
- Organização de código
- Lógica de CRUD
- Manipulação de dados

---

## ⚙️ Funcionalidades
- ✅ Cadastro de veículos  
- 📋 Listagem de veículos  
- ✏️ Atualização de dados  
- ❌ Remoção de registros  

---

## 🛠️ Tecnologias utilizadas
- Python
